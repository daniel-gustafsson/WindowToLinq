BeginAggregate<(Of <<'(TSource>)>>)
This is an alias for a window on an input sequence that is optimized for multi aggregate calculation. The window has no preceding bound and ends at current row. That way multiple aggregates can be computed for all the elements of the input sequence without any buffering, and by fetching just the last value you simply get all aggregates from a single pass through the sequence.

LastOrDefault<(Of <<'(TSource, TSourceAggregates>)>>)
A special version of LastOrDefault that returns the last value in a aggregated window sequence. If the sequence is empty, a tuple is returned with the source value as default, and all aggregates set to the results based on their seeds. Instead of calling this function directly, it is recommended ot use one of the Compute overloads instead.
