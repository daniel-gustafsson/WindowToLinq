Remarks

In this extension to Linq to Objects, a window is defined as a subsequence of a source sequence that is specified by the boundary functions associated with the window. The boundary functions can operate on the values and/or the index of the values of the source sequence. The window can then be operated on as a sequence itself or can be used in an optimized way with specific aggregate functions. The latter is especially useful if operating on an unending input source.
